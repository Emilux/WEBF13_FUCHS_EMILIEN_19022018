LES LISTES

Listes non ordonnées

Pour faire une liste non ordonné on utilise la balise UL pour la commencer et chaque élément de la liste sera mit derrière une balise ouvrante LI. Par defaut chaque élément de la liste est marqué d'un cercle plein.

Exemple :

  • Basket
    • Balle de basket
    • Maillot de basket
    • Chaussure de sport
  • Football
    • Balle de foot
    • Maillot de foot
    • Chaussure à crampon
  • Handball
    • Balle de handball
    • Maillot de handball
    • Chaussure de sport

Remarque : Il est possible de modifier l'apparence du point de la liste UL graçe à du CSS en ecrivant directement dans la balise (style="list-style-type:le_type_de_forme"), les formes sont les suivantes :

Valeur Description
SQUARE Affiche une forme de carré à chaque lignes
CIRCLE Affiche une forme de cercle vide à chaque lignes
DISC Affiche une forme de cercle plein à chaque lignes (par défaut)
NONE N'ffiche aucune forme à chaque lignes

Le style CSS peut aussi être rajouté via un fichier CSS externe.

Listes ordonnées

Pour faire une liste non ordonné on utilise la balise OL pour la commencer et chaque élément de la liste sera mit derrière une balise ouvrante LI. Par defaut chaque élément de la liste est marqué d'un nombre.

Exemple :

  1. Basket
  2. Football
  3. Handball

Remarque : Il est possible de modifier l'apparence des nombres de la liste OL graçe à l'attribut type placé dans la balise OL Les différents types de marqueurs sont les suivants :

TYPE Description
Type="1" Par defaut. Affiche des nombres à chaque lignes
Type="A" Affiche des lettres majusucles à chaque lignes
Type="a" Affiche des lettres minuscules à chaque lignes
Type="I" Affiche des chiffres Romain en majuscule à chaque lignes
Type="i" Affiche des chiffres Romain en majusucles à chaque lignes

Listes de définitions

La listes de définitions est une liste de termes qui possèdent chacun leur définition. La liste de définitions se commence avec une balise DL ensuite est suivi par la baliste DT qui permet d'entrer le terme et enfin la balise DD qui permet d'ecrire la description du terme.

Exemple :

Football
Courir derrière une balle
Sur un terrain en herbe en pleine
il faut mettre la balle dans le but adverse
Handball
Courir avec une balle en main
Dans une salle
il faut mettre la balle dans le but adverse
Basket
Dribler avec une balle en main
Dans une salle
il faut mettre la balle dans le panier adverse
