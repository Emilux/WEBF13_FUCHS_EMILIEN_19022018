La balise "FORM" a comme attribut obligatoire la method (à post ou get) l'action qui est la cible (la page html à laquelle on envoie les données du formulaire).

  • Les attributs du de la balise FORM
    • Method
      • get
      • post
    • action
    • Name
  • Les attributs de la balise INPUT
    • Type
      • Text
        Permet à l'utilisateur d'entrer du texte(crée une zone de texte).
      • Password
        Permet à l'utilisateur d'entrer du texte caché par des étoiles.
      • Required
        Affiche, quand l'utilisateur appuie sur validé, si le champ qui à l'attribut "Required" est rempli ou pas.
      • Radio
        Permet à l'utilisateur de faire un choix entre plusieurs option graçe à des boutons Radio.
      • Checkbox
        Permet à l'utilisateur de cocher plusieurs option graçe à des checkbox.
      • email
        Permet à l'utilisateur d'entrer une adresse e-mail et le formulaire vérifie directement s'il s'agit bien d'un e-mail.
      • date
        Permet à l'utilisateur de sélectionner une date.
      • url
        Permet à l'utilisateur d'entrer un URL et le formaile vérifaire directement s'il s'agit bien d'une URL.
      • submit
        Crée une bouton valider et récupère toute les infos du formulaire.
      • Reset
        Crée une bouton réinitialise et supprime toute les infos du formulaire.
      • Progress
        Crée barre de progression.
      • image
        Crée une bouton valider sous forme d'image.
    • Name
      Donne une nom au input crée se qui servira plus tard pour récupérer un certaine valeur.
    • Size
      Change la taille de l'input crée
    • Value
      Le value pour un input de type "text" permet d'y assigner un certain texte sinon il permet d'identifier une certaine valeur.
    • Autofocus
      Permet de séléctionner automatiquement un certain input.
    • placeholder
      Permet d'assigner un texte en arrière plan de l'input (d'une zone de texte).
    • Checked
      Le checked est utile pour les bouton radio et checkbox, il permet de selectionner une valeur par défaut.
    • rows
      Permet de modifier la hauteur d'une zone de texte.
    • Cols
      Permet de modifier la largeur d'une zone de texte.
  • Les élément de la balise SELECT
    • La balise OPTION
      Permet de rajouter des selections dans le SELECT, en ajoutant l'attrubut "selected" dans la balise ouvrante d'une des
      options, elle sera sélectionné par defaut.
  • Les élément de la balise DATALIST
    • Input list="nom_datalist"
      Avant de crée une datalist il faut toujours mettre une balise input avec un attribut list="nom_datalist" cela permet de
      créer la datalist.
    • ID
      L'attribut ID permet de récupéré input list="nom_datalist" afin de savoir graçe à qu'elle input la datalist est crée.
    • La balise OPTION
      Permet de rajouter des options à la datalist et afin de chacune les nommers il faut utiliser l'attribut value="nom_option"
