MULTIMEDIA

Mettre un fichier audio

Afin d'ajouter un fichier audio à la page il faut mettre une balise "AUDIO"
Avec l'attribut controls vous pourrez afficher le player de votre fichier comme ci-dessous
afin de gérer plusieurs type de fichier (.mp3, .ogg, .wav) on rajoute une balise "SOURCE" dans "AUDIO"
et avec l'attribut "SRC" donner le lien de votre fichier audio avec n'importe qu'elle extension audio.
le lecteur choisira donc automatiquement le fichier audio compatible avec le navigateur(le premier
étant fonctionnel)..
